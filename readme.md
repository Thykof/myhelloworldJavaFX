# myhelloworldJavaFX
Simple java fx app

[![Build Status](https://travis-ci.org/Thykof/myhelloworldJavaFX.svg?branch=master)](https://travis-ci.org/Thykof/myhelloworldJavaFX)


# Run

`gradle run`

# Create jar

`gradle jar`

## Run the jar

`java -jar build/libs/hellofxfourth.jar`
